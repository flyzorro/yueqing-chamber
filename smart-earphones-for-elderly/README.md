# 老年人智能耳塞项目

## 🎯 项目目标
开发一款针对老年人的智能助听耳塞，主打"听得清，用着简"。

## 📋 核心功能
- **智能助听**：双麦克风波束成形，聚焦前方人声
- **蓝牙连接**：支持手机连接、电话接听
- **长续航**：8小时连续使用 + 24小时充电仓续航
- **简单操作**：一键控制，语音提示

## 🏗️ 技术方案
**主芯片**：WQ7036A (物奇微电子)
- 蓝牙5.4 + BLE Audio
- HiFi5 DSP + NPU
- 支持8路麦克风输入
- 集成ANC/ENC降噪

## 💰 成本目标
- **BOM成本**：¥187 (一对耳机+充电仓)
- **建议零售价**：¥299
- **渠道价**：¥199-229
