# 老年人智能助听耳塞 - 芯片筛选报告

## 筛选标准
基于产品核心需求：**ENC（环境降噪） + 波束成形（Beamforming）**

### 必须满足的条件
1. **支持 ENC 算法** - 通话时让对方听清
2. **支持双麦克风波束成形** - 助听核心功能
3. **内置 DSP** - 实时音频处理
4. **成本控制** - 芯片价格 < ¥20

### 可选条件
- ANC（主动降噪）：非必需，可去掉以降低成本
- 蓝牙版本：5.0以上即可
- 其他高级功能：非必需
